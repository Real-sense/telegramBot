Ф-ционал бота
------------

1. Бот отправляет случайные фотки по тегу
2. Бот отправляет курс валюты в HTML с уведомлением


Инструменты:
1. NodeJS
2. NPM

Модули:
1. fs - работа с файловой системой
2. request - работа с запросами
3. lodash - библиотека с заготовленными ф-циями
4. Telegram bot api - готовый фреймворк для работы с апи телеграмма

В файле config.js хранится токен бота от @botFather

Ссылки:
1. [Telegram Api](https://core.telegram.org/methods)
2. [Fixer io/Курсы валют](http://fixer.io/)
